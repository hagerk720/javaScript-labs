## Lab 9 
** Alarm project **
user can set list of alarms and save it in his **local storage**
and remove any alarm if he want 

```javascript
class Alarm {
  constructor(id, hour, minute, sec) {
    this.id = id;
    this.hour = hour;
    this.minute = minute;
    this.sec = sec;
  }
}
```
